Overview

Arrow application layer provides users the ability to create decentralized financial markets (DFMs) around user specified on-chain or off-chain assets (underliers). The primitives of a DFM consist of European style options and are standardized by a decentralized aggregate counterparty (DAC) which provides safety guarantees for settlement of financial obligations in the DFM.
